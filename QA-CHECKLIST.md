# QA Checklist

## Onboarding

- [ ] Required fields validate correctly.
- [ ] Approximate birth time works when exact time is unknown.
- [ ] Main interest options match spec.
- [ ] Optional fields can be skipped.
- [ ] Cosmic profile is created.
- [ ] Copy is magical, simple, and not excessive.

## Daily Cards

- [ ] Card includes word, message, number, color, energy, and advice.
- [ ] Card begins with positive tone.
- [ ] No deterministic or fear-based language appears.
- [ ] Save works.
- [ ] Free and paid behavior matches monetization.

## Weekly Predictions

- [ ] Includes all required sections.
- [ ] Starts with something positive.
- [ ] Love, work, money, and wellness are safe.
- [ ] Money does not provide financial certainty.
- [ ] Wellness does not provide medical claims.

## AI Chat

- [ ] Allowed topics produce useful reflective answers.
- [ ] Forbidden topics redirect safely.
- [ ] Chat uses relevant user context.
- [ ] Chat does not expose system prompt or secrets.
- [ ] Chat history respects preferences.

## Palm Reading

- [ ] Left and right hand upload/capture works.
- [ ] Photo instructions are clear.
- [ ] Poor image quality path is handled.
- [ ] Reading includes allowed palm sections.
- [ ] No death, illness, tragedy, accidents, danger, or medical claims.
- [ ] Images are private.

## Compatibility

- [ ] Person A and Person B required fields validate.
- [ ] Optional photos/avatars work.
- [ ] Result includes percentage and all required sections.
- [ ] No cheating, betrayal, doom, danger, soulmate certainty, or leave/stay instruction.
- [ ] Result is saved.

## Paywall

- [ ] Free, Plus, Pro, and one-time purchases are clear.
- [ ] Restore purchases is visible.
- [ ] Terms and privacy placeholders are visible.
- [ ] Subscription duration and price are clear.
- [ ] Paywall does not use fear-based upsell copy.

## Sharing

- [ ] Native share sheet works.
- [ ] Save as image works.
- [ ] Instagram Story format exports correctly.
- [ ] Instagram Post format exports correctly.
- [ ] WhatsApp sharing works through native share.
- [ ] Watermark behavior follows entitlement.
- [ ] Share event is tracked.

## Notifications

- [ ] Permission prompt appears at appropriate time.
- [ ] Daily reminder can be enabled and disabled.
- [ ] Weekly reminder can be enabled and disabled.
- [ ] Copy is positive and non-alarming.
- [ ] Preferences persist.

## Profile

- [ ] Profile photo/avatar works.
- [ ] Birth information displays correctly.
- [ ] Saved cards display.
- [ ] Saved palm readings display.
- [ ] Saved compatibility readings display.
- [ ] Chat history displays if enabled.
- [ ] Subscription status displays.
- [ ] Settings are editable.

## Safety Content

- [ ] App disclaimer appears in required surfaces.
- [ ] Death prompts are redirected.
- [ ] Cheating/betrayal certainty prompts are redirected.
- [ ] Medical diagnosis prompts are redirected.
- [ ] Pregnancy prediction prompts are redirected.
- [ ] Financial certainty prompts are redirected.
- [ ] Curse/dark threat prompts are redirected.
- [ ] Relationship command prompts are redirected.

## Cross-Platform iOS

- [ ] App launches.
- [ ] Auth works.
- [ ] Camera permissions work.
- [ ] Photo permissions work.
- [ ] Notifications work.
- [ ] Purchases and restore work in sandbox.
- [ ] Sharing works.

## Cross-Platform Android

- [ ] App launches.
- [ ] Auth works.
- [ ] Camera permissions work.
- [ ] Photo permissions work.
- [ ] Notifications work.
- [ ] Purchases and restore work in sandbox.
- [ ] Sharing works.

## Accessibility

- [ ] Tap targets are large enough.
- [ ] Text contrast is readable.
- [ ] Dynamic type does not break key screens.
- [ ] Icons have labels where needed.
- [ ] Motion is subtle and non-flashing.

## Performance

- [ ] Home loads quickly.
- [ ] Cached daily card opens without repeated AI call.
- [ ] Weekly prediction caching works.
- [ ] Palm image upload shows progress.
- [ ] AI calls show clear loading state.
- [ ] Share card generation does not freeze app.

## Error States

- [ ] Network errors are clear and calm.
- [ ] AI failure has safe fallback copy.
- [ ] Payment failure does not unlock access.
- [ ] Restore failure is understandable.
- [ ] Upload failure allows retry.
- [ ] Permission denial offers settings guidance.
